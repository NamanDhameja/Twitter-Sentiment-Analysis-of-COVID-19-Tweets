There are few file in this repository:
1. Pickled_Algo folder contains all the modules specially designed for sentiment analysis.
2. Short_reviews folder contains all file containing positive and negative words, phrases.
3. Create_module file: Creates module files for easy and faster usage of algorithms.
4. sentiment_mod file: Used to implement all the created modules using the later file.
5. twitterapi file: Credentials of twitter developer account
6. gather_data: Used to gather filtered covid19 data from twitter.
7. graph_data: Used to design graph according to the sentiment read by the files.
